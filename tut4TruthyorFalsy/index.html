Truthy and Falsy Values

Truthy Value: A value that evaluates to true in a boolean context. Examples include non-zero numbers, non-empty strings, and objects.
Falsy Value: A value that evaluates to false in a boolean context. Examples include false, 0, "" (empty string), null, undefined, and NaN.

Falsy Values

  • 0
  • -0
  • NaN
  • '' (empty string)
  • undefined
  • null
  • false
