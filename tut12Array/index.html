Arrays

Basic Array Methods

  1. pop() – last element ko remove karne ke liye use hota h
  2. push() – array ke end me new element add karne ke liye use hota h
  3. shift() – array ka first element remove karne ke liye use hota h
  4. unshift() – array ke starting me new element add karne ke liye use hota h
  5. concat() – do ya jyada arrays ko jodne (merge) ke liye use hota h
  6. indexOf() – kisi value ka index find karne ke liye use hota h
  7. includes() – check karne ke liye ki value array me exist karti h ya nahi
  8. reverse() – array ko ulta (reverse order) karne ke liye use hota h
  9. sort() – array ko ascending ya alphabetical order me arrange karne ke liye
  10. slice() – array ka copy part nikalne ke liye without original change
  11. splice() – array me add, delete ya replace karne ke liye (original change hoti h)
  12. Join() array me jo elments unke bech me values add arta h
