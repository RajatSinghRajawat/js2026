JavaScript Variable Scope - Theory

1. Global Scope

Global scope ka variable poori file me accessible hota hai. Function ke bahar declare kiya jata hai.

2. Function Scope

Function ke andar declare variable sirf us function me use hota hai. Mostly var function scope follow karta hai.

3. Block Scope

ES6 me let aur const block scope follow karte hain. Curly braces { } ke andar hi accessible rehte hain.
